Class implements simple linear regression.

Simple linear regression is a linear regression model with a single explanatory variable. The parameters of the regression are calculated by minimizing the OLS. In other words class computes the best-fit linear regression coefficients of the model for the dataset (x, y) by minimizing the sum:
