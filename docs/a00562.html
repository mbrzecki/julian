Function saves curve in DataFrame.

Function calculates and saves curve using following representation:

  • Discount Factors
  • Zero coupon rates
  • 1 day fwd rates approximating instantaneous rate
  • forward rates with basis: 1W, 2W, 1M, 2M, 3M, 6M
